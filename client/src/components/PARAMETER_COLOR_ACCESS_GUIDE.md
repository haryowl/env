# How to Access Individual Parameter Color Customization

## Quick Access Methods

### Method 1: Settings Page (Recommended)
1. **Navigate to Settings**:
   - Click on your profile/avatar in the top-right corner
   - Select "Settings" from the dropdown menu

2. **Find Parameter Colors Section**:
   - In the Settings page, look for the "Appearance" section
   - You'll see a **"Parameter Colors"** card with current color status
   - Click **"Customize"** button to expand the options

3. **Quick Color Schemes**:
   - **Default**: Original color scheme
   - **Ocean**: Blue and orange combinations  
   - **Forest**: Green and red combinations
   - Click any scheme to apply instantly

4. **Advanced Customization**:
   - Click **"Advanced"** button to open the full Parameter Color Customizer
   - Use individual color pickers for each parameter
   - Real-time preview of changes

### Method 2: Direct Parameter Color Customizer
1. **Navigate to Data Menu**:
   - In the sidebar, go to **Data** section
   - Click on **"Parameter Colors"**

2. **Full Customization Interface**:
   - Choose from predefined color schemes
   - Use individual color pickers for each parameter
   - Real-time preview of all changes
   - Reset to default colors anytime

### Method 3: Full Color Customizer
1. **Navigate to Data Menu**:
   - In the sidebar, go to **Data** section  
   - Click on **"Color Customizer"**

2. **Switch to Parameter Colors Tab**:
   - Click on the **"Parameter Colors"** tab
   - Access all parameter color customization options
   - Also customize theme colors in the same interface

## What You'll See

### Current Color Status
- **Color chips** showing each parameter's current color
- **Status alerts** indicating if custom colors are active
- **Real-time updates** when colors change

### Parameter Color Options
- **pH Level**: Purple (#8B46C1) - Acidity/Alkalinity Level
- **COD**: Orange (#F59E0B) - Chemical Oxygen Demand
- **TSS**: Green (#10B981) - Total Suspended Solids  
- **NH3N**: Pink (#EC4899) - Ammonia Nitrogen
- **Debit**: Blue (#3B82F6) - Flow Rate
- **Speed**: Gray (#6B7280) - Velocity

### Where Colors Appear
- **Dashboard**: Parameter Overview section with KPI cards
- **Theme Demo**: Interactive demonstration page
- **Parameter Demo**: Dedicated parameter color showcase
- **All KPI Cards**: Throughout the application

## Troubleshooting

### If You Don't See Parameter Colors
1. **Check Theme**: Make sure you're using any theme (not just KIMA)
2. **Refresh Page**: Try refreshing the browser
3. **Clear Cache**: Clear browser cache and reload
4. **Check Console**: Look for any JavaScript errors

### If Colors Don't Apply
1. **Check localStorage**: Ensure colors are saved properly
2. **Try Different Browser**: Test in a different browser
3. **Reset to Default**: Use the reset button and try again
4. **Contact Support**: If issues persist

### If Settings Don't Show Parameter Colors
1. **Update Settings**: The Parameter Colors section should now be visible
2. **Check Permissions**: Ensure you have access to settings
3. **Try Direct Access**: Use the Data → Parameter Colors menu

## Quick Start Guide

1. **Go to Settings** → **Appearance** section
2. **Find "Parameter Colors"** card
3. **Click "Customize"** to expand options
4. **Try a preset** like "Ocean" or "Forest"
5. **Click "Advanced"** for full customization
6. **See changes** immediately in Dashboard

## Pro Tips

- **Use presets first** to get familiar with the system
- **Customize individual parameters** for fine-tuning
- **Test different combinations** to find what works best
- **Save frequently** - colors are automatically saved
- **Reset anytime** if you want to start over

The individual parameter color system is now fully accessible and working across all themes!
